# 📝 Kanban Pro

![Demo del Proyecto](ruta-a-tu-gif-animado-o-captura.gif)

## 📖 Descripción general
"Kanban Pro es una herramienta de gestión de tareas Fullstack que permite a los usuarios crear, mover y organizar tarjetas en diferentes columnas. El enfoque principal fue implementar un sistema de autenticación seguro y relaciones de bases de datos."]

🔗 **[¡Prueba la aplicación en vivo aquí!](Enlace-a-tu-demo-en-vivo)**

## 🛠️ Tecnologías utilizadas
* **Frontend:** HTML, CSS, JavaScript (Manipulación del DOM).
* **Backend:** Node.js, Express.js.
* **Base de datos:** PostgreSQL, Sequelize.
* **Autenticación:** JWT, Bcrypt.

## ⚙️ Instalación y Uso (Despliegue local)
Para correr este proyecto en tu máquina local, sigue estos pasos:

1. **Clona el repositorio:**
   ```bash
   git clone [https://github.com/ipando86/Ejercicio-Final-EF-M8.git](https://github.com/ipando86/Ejercicio-Final-EF-M8.git)
Navega al directorio:

Bash

cd Ejercicio-Final-EF-M8
Instala las dependencias:

Bash

npm install
Configura las variables de entorno:
Crea un archivo .env en la raíz del proyecto y añade las variables necesarias (Ej: credenciales de DB, Secret Key de JWT).

Ejecuta la aplicación:

Bash

npm run start
